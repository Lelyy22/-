С каким инструментом Git  поставляется? 
А.git name; 
B. git config;
C.git clone.

Что первым нужно сделать при установке Git?
А. получать и устанавливать переменные конфигурации. 
В.создать git который содержит все необходимые файлы репозитория.
С.указать свое имя пользователя и адрес электронной почты.

Каталог Git это? 
А. это файл, обычно содержащийся в вашем каталоге Git, в котором хранится информация о том, что войдет в ваш следующий коммит. 
В.это единая проверка одной версии проекта.
С.это место, где Git хранит метаданные и базу данных объектов для вашего проекта.
